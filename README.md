## How to Run Locally

1. **Clone the repository:**

   ```
   git clone <your-repo-url>
   cd product-shop
   ```

2. **Install dependencies:**

   ```
   npm install
   ```

3. **Start the development server:**

   ```
   npm run dev
   ```

4. **Open in your browser:**
   ```
   http://localhost:5173
   ```
